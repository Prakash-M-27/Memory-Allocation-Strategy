🧠 Project Title

Memory Allocation Strategy Simulator (First Fit, Best Fit, Worst Fit)

---

📖 Description

This project simulates three common memory allocation strategies used in operating systems:

- First Fit
- Best Fit
- Worst Fit

The simulator allocates memory blocks to processes and analyzes:

- Memory utilization
- Internal fragmentation

It helps in understanding how different allocation strategies affect system performance and resource usage.

---

🎯 Objectives

- To implement First Fit, Best Fit, and Worst Fit algorithms
- To compare memory allocation efficiency
- To analyze fragmentation in each method
- To study how memory is utilized in different strategies

---

⚙️ Technologies Used

- Programming Language: Python 3
- Platform: Any system with Python installed

---

🏗️ How It Works

1. Define memory block sizes
2. Define process sizes
3. Apply allocation strategies:
   - First Fit → allocates first suitable block
   - Best Fit → allocates smallest suitable block
   - Worst Fit → allocates largest available block
4. Calculate:
   - Memory utilization (%)
   - Fragmentation
5. Display results in tabular format

---

▶️ How to Run the Program

1. Install Python (if not already installed)
2. Copy the code into a file named:
   memory.py
3. Open terminal or command prompt
4. Run the program:
   python memory.py1
